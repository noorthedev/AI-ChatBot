🤖 AI Chatbot Assistant

A modern AI-powered chatbot built using HTML, CSS, and JavaScript with a clean glassmorphism user interface. The application enables real-time conversations through the Anthropic Claude API, providing intelligent responses in both Urdu and English.

✨ Features:
• Real-time AI chat interface
• Anthropic Claude API integration
• Urdu and English language support
• Conversation history management
• Typing indicator animation
• Auto-resizing message input
• Responsive glassmorphism design
• Enter-to-send and Shift+Enter support
• Error handling and API validation

🛠 Technologies Used:
• HTML5
• CSS3
• JavaScript (ES6)
• Fetch API
• Anthropic Claude API

This project demonstrates API integration, asynchronous JavaScript, chat interface design, conversation state management, and modern frontend development practices.
